# 运营方案 - 男友相机

## 1. 小红书推广策略

### 目标
- 早期：精准触达目标用户（小红书 18-35岁女性）
- 长期：自然增长，口碑传播

### 选题方向

#### 方向一：教男友系列（最高共鸣）
- "让男友学摄影的第1天"
- "教男友拍照技巧，他终于开窍了！"
- "用这个工具，男友拍照从2分到8分"

#### 方向二：吐槽男友系列（高互动）
- "男友镜头下的我vs我眼中的自己"
- "吐槽男友拍照技术，笑着笑着哭了"
- "同一地点，男友拍 vs 我拍"

#### 方向三：工具测评系列
- "发现一个超准的AI拍照分析工具！"
- "上传男友拍的照片，被点评笑到了哈哈哈哈"
- "这个AI比我还会夸男朋友"

#### 方向四：情侣互动系列
- "和男友一起玩拍照分析，笑到肚子疼"
- "男朋友看到分析结果后..."
- "用游戏的方式教男友拍照"

## 2. 内容策略

### 标题公式
- 情绪词 + 具体场景 + 悬念/反转
- 例："这个AI把我男友的照片夸疯了哈哈哈哈"

### 配图
- 真实照片分析截图（有隐私保护的）
- 对比图（Before/After）
- 聊天记录截图

### 标签
- #男友拍照 #情侣日常 #拍照技巧
- #AI工具 #修图教程 #摄影入门
- #小红书运营 #种草

## 3. 引流路径

```
小红书笔记 ↓
    ↓（评论区引导）
点击链接 → 男友相机小程序/H5
    ↓
用户上传照片 → AI分析 → 分享结果
    ↓
分享到小红书 → 二次传播（UGC循环）
```
