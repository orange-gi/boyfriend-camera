# 技术方案 - 男友相机

## 参考架构

复刻知与行（zhiyuxing）项目架构。

## 1. 技术栈

### 前端 App
- **框架**: Expo + React Native（与知与行完全一致）
- **路由**: expo-router（文件路由）
- **状态管理**: Zustand + React Query
- **存储**: MMKV（本地缓存）+ AsyncStorage
- **AI 图像**: expo-image-picker（选图）+ MiniMax VL（图像理解）
- **包管理**: pnpm

### 后端
- **平台**: 腾讯云 CloudBase（与知与行共用同一环境）
- **Runtime**: Nodejs18.15
- **数据库**: CloudBase 关系数据库（RDB）
- **存储**: CloudBase 存储（照片临时上传，分析后删除）
- **认证**: CloudBase Auth（邮箱验证码 / 密码登录）

### AI 能力
- **图像理解**: MiniMax VL（复用 TOOLS.md 中的 API Key）
- **文字生成**: MiniMax / DeepSeek（复用知与行的 deepseek-json 云函数）

## 3. 新建云函数

### analyze-photo
```
功能: 接收照片，调用 MiniMax VL 分析，组装文案返回
输入: { image_base64: string }
输出: {
  score: number,
  highlights: string[],
  suggestions: { dimension: string, text: string }[],
  tip: string
}
依赖: MiniMax API Key
```
